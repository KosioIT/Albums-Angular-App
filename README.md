# Albums-Angular-App

A full-stack web application built with AngularJS 20 (frontend), Express.js (backend), and MongoDB. Designed for performance, scalability, and clean architecture.

## Technologies

**Frontend**
- Angular 20
- RxJS 7.8
- TypeScript 5.3
- Bootstrap

**Backend**
- Node.js 20
- Express.js
- MongoDB + Mongoose
- JWT, dotenv
